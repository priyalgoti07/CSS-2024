The Position CSS property sets how an elemet is positioned in a documnet

Position: static / relative/ absolue/ fixed

Position Playground

fixed

positioned relative to browser. (removed from flow)

static(byDefault)

default Position(The top, right, bottom,left,and z-index have no effect)

relative

elemnt is relative to itself (The top, right, bottom,left,and z-index will work)

Sticky

Positioned relative ti its closest ancestor(parent element)

static : top,bottom,left,right,z-index not to works
